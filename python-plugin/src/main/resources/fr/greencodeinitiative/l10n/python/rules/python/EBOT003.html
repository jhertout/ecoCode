Using list.append(x) within a loop to copy a list is not energy efficient.

Prefer the usage of list.copy() which is more energy friendly.

Noncompliant Code Example

    list = [1, 2, 3, 4, 5]
    new_list = []
    for item in list:
        new_list.append(item)

Compliant Solution

    list = [1, 2, 3, 4, 5]
    new_list = []
    new_list = list.copy()
