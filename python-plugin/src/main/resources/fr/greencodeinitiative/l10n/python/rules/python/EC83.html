Using list.append(x) within a loop to copy a list is not energy efficient.

Prefer the usage of native methods which are more energy friendly.

To Copy

Noncompliant Code Example

    list = [1, 2, 3, 4, 5]
    new_list = []
    for item in list:
        new_list.append(item)

Compliant Solution

    list = [1, 2, 3, 4, 5]
    new_list = []
    new_list = list.copy()

To Map

Noncompliant Code Example

    list = [1, 2, 3, 4, 5]
    new_list = []
    for item in list:
        new_list.append(item)

Compliant Solution

    def square(number):
        return number ** 2

    list = [1, 2, 3, 4, 5]
    new_list = list(map(square, list))
