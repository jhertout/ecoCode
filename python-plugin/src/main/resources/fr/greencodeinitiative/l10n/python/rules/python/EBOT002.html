Using copy.deepcopy(x) of module copy to perform a shallow copy of a list is not energy efficient.

Prefer the usage of list.copy() which is more energy friendly in that case. If a deep copy is still required, keep copy.deepcopy(x).

Noncompliant Code Example

    import copy

    my_list = [1, 2, 3, 4]
    list_copy = copy.deepcopy(my_list)

Compliant Solution

    my_list = [1, 2, 3, 4]
    list_copy = my_list.copy()
